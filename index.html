About Task Timer Light      License: GPLv3

Clear

Task Timer Light is a simple, lightweight time tracking utility which allows you to start and stop a timer for each task or project you are working on. It will work on your local computer (with no webserver required) if you don't need saving. It is designed for one person's use.

This version has a minimal (and optional) server-side component for permanent saving of your logged times. When you leave or refresh the page, all timers are reset. If saving is enabled, only one instance of the application should be in use at a given time.
